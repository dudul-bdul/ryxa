## Requirements

- Install Nodejs @latest

```shell
# Ubuntu
sudo apt install nodejs
```

```shell
# Windows
sudo choco install nodejs
```

```shell
# asdf
asdf install nodejs latest
asdf global nodejs latest
```

- pnpm

```shell
npm install -g pnpm
```

## Local Run

Install deps

```shell
pnpm i
```

Run dev server @ `localhost:3000`

```shell
pnpm dev
```

## Local Build

```shell
pnpm build
```

```shell
pnpm preview
```

## Build & Deploy onto Netlify

```shell
netlify build
```

```shell
netlify deploy
netlify deploy --prod
```
